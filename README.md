# Customer Churn Prediction Dashboard

An end-to-end Deep Learning project designed to predict the probability of a customer leaving a bank. This repository contains the complete pipeline: from exploratory data analysis and model training using an Artificial Neural Network (ANN), to a fully interactive web dashboard built with Streamlit.

## 🚀 Tech Stack

* **Core:** Python 3.9
* **Deep Learning:** TensorFlow 2.10.1, Keras
* **Data Science:** Pandas, NumPy, Scikit-Learn
* **Web Framework:** Streamlit
* **Visualization:** Plotly (Gauge charts), TensorBoard
